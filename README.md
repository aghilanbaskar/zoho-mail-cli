# Zoho Mail CLI

Zoho mail cli is node client tool used to authenticate and read account and messages data

## Prerequisites
    * npm  >= 5.0.0
    * node >= 10.0.0

## Installation


```bash
git clone https://github.com/aghilanbaskar/zoho-mail-cli.git

cd zoho-mail-cli

npm i

npm install -g .

```

## Usage

To authenthicate the cli app.
```cli
mail login
```
To read the messages.
```cli
mail message
```
